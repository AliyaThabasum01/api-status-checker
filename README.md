# 🌐 API Status Checker

A simple Python CLI tool to check if a website or API is online.

## ✨ Features

- Check website/API availability
- Display HTTP status code
- Measure response time
- Beginner-friendly

## 📦 Installation

```bash
pip install -r requirements.txt
```

## ▶️ Run

```bash
python main.py
```

## 📌 Example

Input:

```
github.com
```

Output:

```
URL           : https://github.com
Status        : ✅ UP
Status Code   : 200
Response Time : 0.187 seconds
```

## 🛠 Built With

- Python
- Requests
